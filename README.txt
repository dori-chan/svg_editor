SVG EDITOR

Версия 1.4
Автор: Овсянникова Дарина

Описание:
Векторный графический редактор
Реализовано семь векторных примитивов: линия,
эллипс, прямоугольник, треугольник + сердце, плюс, звезда
Доступны 5 цветов для заливки и обводки: черный, белый, красный,
зеленый, голубой 


Дополнительный функционал: очистить холст, градиент

Доп. задачи:
1) Возможность сохранять изображение в 
формате SVG 
2) Возможность открывать ранее сохраненные SVG-изображения + добавлена обводка фигур
3) Добавлены новые примитивы + градиент
